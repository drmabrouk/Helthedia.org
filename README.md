# Helthedia

**Domain:** helthedia.org

## Purpose
Helthedia is a specialized search engine designed exclusively for **scientific, medical, and health-related research**. Its main goal is to provide researchers, doctors, students, and institutions with **reliable, peer-reviewed, and indexed studies** in one centralized platform.

## Target Users
- Healthcare professionals
- Medical students
- Research institutions
- Anyone seeking authoritative scientific information

## Core Concept
The platform functions like a “Google for medical research,” allowing **advanced search, filtering, and sorting** of research papers, clinical trials, journals, and systematic reviews.

## Design Philosophy
Clean, professional, and minimalistic interface with a **black, white, and grayscale color palette**, fully **responsive for all devices**, emphasizing usability, readability, and accessibility.

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
